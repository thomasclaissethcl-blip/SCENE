Principe général

Vous connaissez les Livres dont vous êtes le héros ? Un parcours interactif fonctionne comme une histoire à embranchements. À chaque étape, une scène présente une situation, puis propose un ou plusieurs choix. Selon la décision prise, le parcours se poursuit vers une autre scène et peut produire des effets visibles ou invisibles.

Les scènes sont les moments du récit. Elles décrivent une situation, donnent du contexte, puis invitent l’utilisateur à agir. Les choix servent à faire avancer l’histoire, mais aussi à personnaliser le chemin suivi.

En arrière-plan, le parcours mémorise certaines informations. On peut les voir comme de petites jauges ou comme des repères invisibles qui évoluent au fil des décisions. Par exemple, un niveau de risque peut augmenter, un climat de confiance peut se dégrader, ou un objectif peut être considéré comme atteint.

Ces informations permettent ensuite de conditionner la suite. Une scène, un événement ou une action peuvent apparaître seulement si certaines conditions sont remplies. Autrement dit, le parcours ne réagit pas seulement au dernier clic, mais à l’ensemble de ce qui s’est construit depuis le début.

On peut résumer le principe ainsi : les scènes proposent des choix, les choix modifient l’état du parcours, et cet état influence la suite de l’histoire.

Dans ce type de structure, deux personnes peuvent donc partir de la même scène et vivre ensuite des parcours différents. C’est ce qui permet de créer une expérience plus engageante, plus crédible, et plus proche d’une situation réelle.

Comprendre le workflow

Deux outils complémentaires sont proposés ici : un générateur de templates, et un game designer.

  1. Vous pouvez commencer par le générateur de template si vous voulez produire rapidement une structure initiale, un canevas de contenu ou une base de départ que vous enrichirez ensuite dans le game designer.
  2. Vous pouvez ensuite passer dans le game designer pour construire le parcours complet, organiser les scènes, les choix, les variables, les règles d’effondrement et les diagnostics (tout cela est expliqué dans l'outil).
  3. Comment choisir ?
    • Si vous savez déjà précisément ce que vous voulez concevoir, vous pouvez ouvrir directement le game designer.
    • Si vous avez d’abord besoin d’un cadre ou d’une trame, il est plus pertinent de commencer par le générateur de template : générez un scénario générique en indiquant au générateur vos préférences, exportez ce scénario, puis importez-le dans le game designer.
