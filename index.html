Comment vous pouvez organiser votre travail

Vous pouvez commencer par le générateur de template si vous voulez produire rapidement une base de départ, une trame ou une structure initiale avant d’entrer dans un travail de conception plus détaillé.

Vous pouvez ensuite utiliser le serious game designer pour transformer cette base en parcours interactif, en organisant les scènes, les choix, les variables, les règles d’effondrement et les diagnostics.

Si votre scénario est déjà clair dans votre esprit, vous pouvez aussi entrer directement dans le serious game designer. Le générateur de template est surtout utile lorsque vous avez besoin d’un premier cadrage avant de construire le jeu lui-même.
