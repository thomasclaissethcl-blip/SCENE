Interruptions et remédiations

Les interruptions, les scènes d’analyse, les remédiations associées et les variables techniques nécessaires sont générées automatiquement par le moteur à partir du type de structure choisi et du nombre de scènes d’analyse demandé.

Dans ce générateur, vous ne paramétrez pas les règles d’interruption ni les variables techniques. Vous préparerez ensuite ces éléments plus finement dans le builder, à partir du template généré.
